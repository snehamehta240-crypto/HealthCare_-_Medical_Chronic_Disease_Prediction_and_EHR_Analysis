# HealthCare_-_Medical_Chronic_Disease_Prediction_and_EHR_Analysis
Healthcare Disease Prediction &amp; EHR Analysis using Python, Machine Learning, Data Analysis, and Visualization.

##  Project Overview
This project evaluates a healthcare prediction model using patient risk scores and clinical data.  
The Power BI dashboard provides **Clinical Evaluation & Patient Risk Overview** with interactive visuals and performance metrics.

##  Key Features
- 📊 **Summary Metrics:** Total Patients, CKD Patients %, High‑Risk Patients %, Avg Risk Score  
- 🧮 **Confusion Matrix:** Model classification performance (TP, TN, FP, FN)  
- 📈 **ROC Curve:** True Positive Rate vs False Positive Rate  
- 👩‍⚕️ **Patient Details Table:** Age, Gender, Disease, Actual, Prediction, Risk Score, Risk Level (color coded)  
- 🎛️ **Interactive Filters:** Disease Types, Age Slicer

Key Insights
Total Patients: 120

CKD Patients: 37% of total population

High‑Risk Patients: 35.8% identified as high risk

Average Risk Score: 0.36 (moderate overall risk)

 Model Performance:-
 
True Negatives: 76 (healthy patients correctly classified)

True Positives: 44 (CKD patients correctly detected)

False Negatives: 44 (CKD patients missed by the model)

False Positives: 0 (no healthy patient misclassified)

Disease Distribution: Balanced across bgr, bp, dm, hemo.

Risk Levels: Majority patients fall into high‑risk category.

ROC Curve: Strong performance in avoiding false positives, but recall improvement is needed.
